Back-End Developer

Reference Number: ABC56

Position Title: Back-End Developer

Brief Description: Responsible for developing and maintaining the server-side logic, databases, and APIs to ensure efficient and scalable web applications.

Salary Range: $80,000 - $110,000 per year

Reports To: Software Development Manager

Key Responsibilities
  • Develop and maintain server-side applications and services
  • Write clean, scalable, and efficient code
  • Design and implement databases and APIs to support web applications
  • Collaborate with front-end developers to integrate user-facing elements with server-side logic
  • Ensure data security and protection across various platforms
  • Optimize back-end processes and troubleshoot issues as they arise
  • Participate in code reviews and maintain code quality standards

Qualifications

Essential:

  • Bachelor's degree in Computer Science, Software Engineering, or a related field
  • Proficiency in server-side programming languages such as Node.js, Python, Java, Ruby, or PHP
  • Experience with databases like MySQL, PostgreSQL, or MongoDB
  • Strong knowledge of RESTful API development and web services
  • Experience with version control systems like Git
  • At least 2-3 years of experience as a back-end developer

Preferable:

  • Experience with cloud platforms such as AWS, Azure, or Google Cloud
  • Familiarity with containerization tools like Docker and orchestration tools like Kubernetes
  • Experience with message brokers like RabbitMQ or Kafka
  • Strong problem-solving skills and experience with performance tuning
  • Familiarity with testing frameworks and tools like Jest, Mocha, or Postman
