Front-End Developer

Reference Number: ABC56

Position Title: Front-End Developer

Brief Description: Responsible for developing the user-facing part of web applications, ensuring a smooth, efficient, and responsive experience for users across different devices.

Salary Range: $70,000 - $100,000 per year

Reports To: Software Development Manager

Key Responsibilities
  • Develop visually appealing and responsive user interfaces using HTML, CSS, and JavaScript
  • Collaborate with UX/UI designers to ensure high-quality design implementation
  • Ensure the technical feasibility of UI/UX designs
  • Write clean, efficient, and maintainable code
  • Work with APIs and backend services to fetch and display dynamic content
  • Optimize applications for maximum speed and scalability across multiple browsers
  • Participate in code reviews and maintain code quality standards

Qualifications

Essential:

  • Bachelor's degree in Computer Science, Web Development, or a related field
  • Proficiency in front-end technologies such as HTML, CSS, and JavaScript
  • Experience with modern front-end frameworks like React, Angular, or Vue.js
  • Strong understanding of responsive design and cross-browser compatibility
  • Experience with version control systems like Git
  • At least 2-3 years of experience as a front-end developer

Preferable:

  • Familiarity with state management tools like Redux, Vuex, or Context API
  • Experience with CSS preprocessors like Sass or LESS
  • Knowledge of modern front-end build tools and workflows (e.g., Webpack, Babel)
  • Familiarity with testing frameworks like Jest, Mocha, or Cypress
  • Experience with front-end performance optimization
