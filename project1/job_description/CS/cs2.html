Full Stack Developer

Reference Number: ABC56

Position Title: Full Stack Developer

Brief Description: Responsible for developing and maintaining both front-end and back-end systems, ensuring efficient and high-quality code is delivered for a seamless user experience.

Salary Range: $80,000 - $110,000 per year

Reports To: Software Development Manager

Key Responsibilities
  • Develop both client-side (front-end) and server-side (back-end) web applications
  • Collaborate with UX/UI designers to create seamless and responsive user interfaces
  • Write reusable, testable, and efficient code
  • Work with databases, APIs, and web services to integrate various functionalities
  • Ensure cross-platform optimization and maintain performance of applications
  • Participate in code reviews and maintain code quality standards

Qualifications

Essential:

  • Bachelor's degree in Computer Science, Software Engineering, or a related field
  • Proficiency in front-end technologies like HTML, CSS, JavaScript, and frameworks like React, Angular, or Vue
  • Experience with back-end technologies such as Node.js, Python, Ruby, or Java
  • Familiarity with relational and non-relational databases (e.g., MySQL, MongoDB)
  • Experience working with RESTful APIs and version control systems like Git
  • At least 3 years of experience in full-stack web development

Preferable:

  • Experience with cloud services like AWS, Azure, or Google Cloud
  • Familiarity with DevOps practices and CI/CD pipelines
  • Strong problem-solving skills and ability to work in an Agile environment
  • Experience with front-end testing frameworks like Jest, Mocha, or Cypress
